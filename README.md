# piezo-pipeline

End-to-end pipeline for piezoelectric signal analysis.

## IMPORTANT

This repository contains scripts exported as `.txt` files.

These scripts are intended to be executed **in Google Colab**.

### How to use

1. Open Google Colab
2. Create a new notebook
3. Upload the desired `.txt` file
4. Copy and paste the content into a Colab cell
5. Run the pipeline

Local execution is **not supported** by design.
